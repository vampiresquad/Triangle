About ▲ Triangle

Triangle is an all-in-one toolkit designed for ethical hackers, penetration testers, and cybersecurity learners.

The goal of this project is to create a fast, lightweight, browser-based toolkit that can perform reconnaissance, encoding, decoding, and payload generation without requiring installation.

Triangle is built as a static web application using HTML, CSS, and JavaScript and hosted on GitHub Pages — ensuring high performance, reliability, and security.
