About ▲ Triangle

Triangle is an all-in-one toolkit designed for ethical hackers, penetration testers, and cybersecurity students. My goal was to create a fast, responsive, and accessible platform that could perform various reconnaissance, encoding, and payload generation tasks directly from the browser.

This project is built entirely as a static site (HTML, CSS, JavaScript) and hosted on GitHub Pages, ensuring maximum speed and security. All tools are client-side and intended for educational and ethical use only.
